Nous pourrions utiliser des mots étranges ou peu familiers tout au long de cette documentation.
Voici leurs définitions (ou du moins ce que nous entendons par là).

## Général

- **Instance** : Un déploiement de la plateforme de covoiturage Karr, géré par des administrateurs. Elle se compose d'un site web frontend, d'une API et d'une base de données.
- **Admin** : Personne/entité qui gère une instance.
- **Configuration vs. Paramètres** : Dans Karr, la Configuration fait référence aux réglages techniques de bas niveau (port API, fournisseurs d'authentification, configuration de la base de données, etc.) ;
  Les Paramètres font référence aux réglages de plus haut niveau pour l'application.<br/>
  La première peut être configurée via le fichier de configuration ou les variables d'environnement, et la seconde est configurée via l'interface web (pas encore implémentée).
